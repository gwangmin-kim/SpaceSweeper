### ✅ 진행 상황
- [x] `Shotgun`/`Bullet`: 다수의  총탄을 부채꼴 형태로 발사. 총탄은 정해진 수명 시간 동안 직선으로 날아가며, 개별적으로 폐기물과 충돌 시 대미지를 입히고 파괴. 관통 옵션 존재. Rigidbody2D 기반 움직임. (Kinematic으로, 초기 운동 상태가 계속 유지됨)
- [x] `Lasergun`: 조준 방향으로 폐기물 대상 탐색. 감지된 대상 직접 타격하여 지속 대미지. 전이 옵션 존재. 전이 횟수만큼 재귀적으로 가장 가까운 폐기물을 탐색하여 동일하게 타격.
- [ ] `SessionManager`: 세션 관리자. 세션 시작/종료 로직, 산소 타이머 관리, 해당 세션 내 수집한 자원 관리. 싱글톤 패턴으로 정적 인스턴스로 접근 가능. 단, 매 세션 새롭게 만들어지며 영구 유지되지 않음.
- [x] `ResourceManager`: 자원/재화 관리자. 싱글톤, 영구적 (`DontDestroyOnLoad`). 세션 정상 종료 시 얻은 자원을 총량에 합산. 자원 소모/재화 획득 (재화로 교환 시), 재화 소모 (업그레이드 시). 세부 로직은 각 소모/획득처에서 구현, 여기선 자원/재화의 양만 관리.
- [ ] `SpaceUIManager`: 탐사 세션에 필요한 UI 정보 출력. 산소량, 획득 자원량, 폐기물 체력 등
- [ ] 에셋 구비 및 탐사 세션 기본 UI 구성
